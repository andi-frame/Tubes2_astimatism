## Getting Started

### Frontend
First add the .env file, then:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Backend
First add the .env file, then:

```bash
go mod download
air
```

For development stage, server is running at [http://localhost:8080](http://localhost:8080)
